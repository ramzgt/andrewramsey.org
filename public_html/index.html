About Me

Software Engineer, Mobile and Web Developer, Database Designer

Also co-founder of the startup Future Life, Inc.

I research, develop, and deploy software applications for a wide variety of applications: business operations, data warehousing, and analyzing scientific laboratory data, for example.

Recently I have been working with Python and the Django framework. In addition I use Bootstrap for mobile apps and PHP/MySQL to build dynamic websites for data-driven applications rapidly. I also have experience developing inside the Microsoft stack using C# for windows client-side applications (WinForms, WPF) as well as ASP.NET MVC applications with MSSQL.

Since 2008, I have been actively researching cross-platform rapid GUI development. I developed a method for providing GUI Rapid Application Design (RAD) for OS-agnostic applications using C++ alongside Qt and PyQT.

Currently I am researching Javascript frameworks focusing on JQuery mobile and AngularJS. I am doing a full-scope study of various MVC frameworks and their advantages and disadvantages to the SDLC.

Other technologies I have worked with: Java, Android, Oracle SQL, ASP.NET MVC, Flowcode, Matlab, and Labview. I have a strong desire to learn the Go language.

During my Bachelor's degree at Georgia Tech I have served as a synthetic organic chemist and materials scientist developing nanoscale thin films for solar cell engineering.
